概要:
- 本ツールは、ShareXのスクロールキャプチャから作成したPDFをOCRし、個人情報をマスクしてOpenAI APIで経歴要約を生成する用途。

主な機能:
- PNGからPDF生成
- OCRによる文字抽出
- 個人情報（メール・電話番号・郵便番号など）のマスク
- OpenAI APIによる要約生成
- クリップボードコピー
- テキスト保存

セットアップ:
1. ShareXをインストール
   - winget install --id ShareX.ShareX -e
2. 仮想環境を作成して有効化
   - python -m venv .venv
   - .venv\Scripts\activate
3. 依存関係をインストール
   - pip install -r requirements.txt

実行方法:
- python summarize_resume.py "PDFパス"

想定フロー:
1) ShareXでスクロールキャプチャ
2) PDF化
3) OCR
4) テキスト抽出
5) 個人情報マスク
6) 要約生成
7) クリップボードへコピー

注意点:
- APIキーはリポジトリに含めない
- 実在候補者のPDFや要約結果はコミットしない

OCR要件:
- ocrmypdf を利用
- 事前に以下を導入
  - Tesseract OCR
  - ocrmypdf
  - jpn.traineddata

Tesseract導入例:
- winget install UB-Mannheim.TesseractOCR
- pip install ocrmypdf

jpn.traineddata:
- 配布元: https://github.com/tesseract-ocr/tessdata/blob/main/jpn.traineddata
- 直接DL: https://raw.githubusercontent.com/tesseract-ocr/tessdata/main/jpn.traineddata
- 配置先: C:\Program Files\Tesseract-OCR\tessdata\
